ScreenSaver

Posted by Magno Correa

Tis is a simple screensaver based on HTML and Bootstrap4 that present in fullscreen some art paints, with the respective name and author.

The configuration is based on a json file, that can be configured to other or more images.

I personaly use in my 3rd monitor when I working at home.
