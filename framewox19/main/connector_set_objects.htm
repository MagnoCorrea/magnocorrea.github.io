How new Connector Set objects differ from the old style connector sets

In response to a unanimous customer vote at Optimize 2002 (The Casewise international user conference) Corporate Modeler now represents Connector Sets as objects in all diagrams.

Note: This functionality effectively means that you create a connector set on any diagrams as you would have done for generic diagrams in Corporate Modeler 9.

As the Connector Set is now an object (just like a Process or an Organization), it can be set to be any shape, to have any appearance settings, or to use any style.

This means that you can create Connector Sets or Decision Diamonds to conform to that of almost any notation. For example: if you are creating an IDEF3 diagram, you can use the special "junction" symbols (provided with Corporate Modeler 10) to make the connector set appear exactly as it should.

About connector set objects

  • A connector set object is created in the New Objects palette from the Object Type "Connector Set".
  • A connector set’s type (what sort of decision it represents) is defined in the palette by its Category.
  • The values (Delay, %split, etc) on the single connector in the set (in the case of an XOR set, the connector which connects to the Connector Set object) are ignored when the diagram is simulated.
  • For the purposes of simulation, connectors that are involved with an XOR Connector Set object must have the category and flow type "Optional".
  • For the purposes of simulation, connectors that are involved with an AND Connector Set object must have the category and flow type "Mandatory".

How will my "old style" connector set objects change in Corporate Modeler 10?

  • A connector set object is drawn in place of the old style connector set.
  • None of the other objects on your diagram change size or position on your diagram.
  • Any data on your connectors is migrated to the new diagram (you will not loose information about %Split, Delay etc).
  • Connector Set objects are created in the New Objects palette for future use (note that editing the Style or Symbol used by the object in the New objects palette will change the appearance of all Connector Sets with that category on the diagram).
